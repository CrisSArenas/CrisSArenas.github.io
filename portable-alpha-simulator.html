Portable Alpha Simulator

Quick Guide — How to Use This Simulator

STEP 1

Understand the Concept

This tool compares a traditional 60/40 portfolio against a portable alpha strategy that uses synthetic beta (futures) + an alpha overlay + collateral yield, minus real-world costs.

STEP 2

Set Benchmark / Beta

Configure the equity/bond split and beta notional (futures exposure). Adjust basis noise and roll cost to model replication frictions.

STEP 3

Configure Alpha Overlay

Set the alpha sleeve's allocation, target vol, IR (expected Sharpe of alpha), beta leakage, and slippage. These define your alpha engine.

STEP 4

Tune Costs & Funding

Set margin requirement, collateral buffer, and financing spread. If total exposure exceeds capacity, financing costs kick in.

STEP 5

Analyze Results

Change any input and the simulation auto-updates instantly. Study the growth chart, drawdown, KPI table, and return decomposition. Click Reset Defaults to restore initial values.

STEP 6

Stress Test

Check the Stress Scenarios table for one-month shocks. Review Simulation Health to verify the model behaves as configured. Hover any dotted label for an explanation.
