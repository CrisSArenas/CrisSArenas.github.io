SLAM Visualizer Guide

What This Tool Does

This visualizer demonstrates Simultaneous Localization and Mapping (SLAM) in a browser. Drive a virtual robot through pre-built indoor environments using WASD keys while watching it build an occupancy grid map in real time using simulated LiDAR scans.

The robot performs raycasting-based LiDAR simulation with 180 beams spanning 360 degrees, and updates the map using log-odds occupancy grid mapping.

SLAM Concepts

Occupancy Grid
A 2D grid where each cell stores a log-odds value representing the probability of being occupied, free, or unknown. Updated via Bresenham-style ray marching.
LiDAR Raycasting
180 beams sweep 360 degrees around the robot. Each beam marches through the environment checking for wall intersections to determine range measurements.
Log-Odds Update
Cells along each ray are marked as free (negative update) while the cell at each hit point is marked as occupied (positive update). This Bayesian approach handles noisy measurements gracefully.
Coverage Mapping
As the robot explores, unknown cells transition to known states. Coverage percentage tracks how much of the navigable environment has been observed.

Panel Descriptions

World View
Top-down rendering of the actual environment with walls, the robot (triangle), LiDAR beams (cyan), and path trail (green).
Occupancy Grid Map
The map being built in real-time. Dark = unknown, dark blue = free space, bright green/cyan = obstacles detected.
LiDAR Scan View
Polar plot of the current 360-degree LiDAR scan with range rings and angular sector labels.
Telemetry
Real-time robot pose, velocity, LiDAR statistics, and map coverage metrics.
