STM32 I/O Lab Guide

What This Tool Does

This interactive lab simulates STM32F446RE memory-mapped I/O registers. Manipulate peripheral registers at the bit level and observe real-time hardware behavior: waveforms, pin states, and equivalent C code.

Peripherals

GPIOA
General Purpose I/O Port A. Configure 16 pins as input, output, alternate function, or analog. Set output levels via ODR.
TIM2
32-bit general-purpose timer. PWM generation with prescaler (PSC), auto-reload (ARR), capture/compare (CCR1).
USART1
Universal serial receiver/transmitter. Configure baud rate (BRR), enable TX/RX, set word length and parity.
ADC1
12-bit analog-to-digital converter. Configure resolution, conversion mode, sampling time.

How to Use

Select Peripheral
Click a tab to load its registers into the editor.
Toggle Bits
Click any bit cell to toggle 0/1. All views update instantly.
Hover for Info
Hover over any bit for field name, description, and current value.
Use Presets
Click preset buttons for common configurations.

Quick Guide

STEP 1

Select Peripheral

Choose GPIO, Timer, UART, or ADC to load its memory-mapped registers.

STEP 2

Toggle Bits

Click any bit to flip it. Hex value, waveform, and code update instantly.

STEP 3

View Waveforms

Oscilloscope shows real-time output: GPIO levels, PWM, UART frames, ADC sampling.

STEP 4

Try Presets

Load common configs: LED Blink, PWM 1kHz, UART 115200, ADC Single.

STEP 5

Read C Code

The code panel shows equivalent CMSIS code that matches your configuration.
